Library Reference

Standard Library

The core set of built-in flows and functions available in the Ouroboros runtime.

I/O Functions

Basic input/output operations.

Print

Prints one or more values to the standard output (console), followed by a newline.

"Hello World" -> Print Print("Value is:", 42)

Flow Control

Utilities for managing flow execution.

Call

Executes a flow or a chain of flows. Useful when you want to trigger a flow for its side effects without capturing a return value, or as the terminator of a forward chain.

// Execute a flow Call(MyFlow) // Chain termination Data -> Process -> Call
