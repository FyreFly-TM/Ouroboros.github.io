Scientific Computing

Domain System

Copy/paste equations directly from textbooks — they just work! Domains enable domain-specific symbol interpretation, constants, and unit systems.

Overview

The Ouroboros Domain System is a revolutionary feature that allows you to write mathematical and scientific equations using their natural notation. Instead of translating formulas into code, you can copy equations directly from textbooks, academic papers, or reference materials.

✨ Key Benefits

  • Natural Notation — Write E = mc² instead of E = m * c * c
  • Domain-Specific Symbols — Each domain knows its standard variable meanings
  • Built-in Constants — Physics constants like c, g, G are predefined
  • Unit Awareness — Pass values with units for clarity
  • Bidirectional Solving — Solve for any variable in an equation

Quick Example

// Define a physics flow using natural notation @Domain.Physics Flow KineticEnergy() E = ½mv² // Use it with named parameters energy := KineticEnergy(m: 10 kg, v: 5 m/s) "Kinetic Energy: " + energy + " J" -> Print // Output: Kinetic Energy: 125 J // Solve for velocity instead! velocity := KineticEnergy(E: 200 J, m: 10 kg) "Velocity: " + velocity + " m/s" -> Print // Output: Velocity: 6.32456 m/s

Calling Domain Flows

Call domain flows using named parameters with optional units:

// With units (recommended for clarity) result := FlowName(param1: value1 unit1, param2: value2 unit2) // Without units (still works) result := FlowName(param1: value1, param2: value2) // Solving for different variables // Just omit the variable you want to solve for! force := NewtonSecondLaw(m: 10 kg, a: 5 m/s²) // Solve for F accel := NewtonSecondLaw(F: 100 N, m: 20 kg) // Solve for a mass := NewtonSecondLaw(F: 100 N, a: 5 m/s²) // Solve for m

How It Works

When you decorate a Flow with @Domain.Name, Ouroboros:

  1. Parses the equation using domain-specific symbol interpretation
  2. Injects constants like g = 9.81, c = 299792458, π = 3.14159...
  3. Recognizes units and handles unit conversions where applicable
  4. Creates a solvable system that can compute any unknown variable

Symbol Resolution Priority

1. Explicitly passed parameters (highest priority) 2. Domain-specific constants (g, c, G, π, e, etc.) 3. Previously defined variables in scope 4. Error if symbol cannot be resolved

Descriptive Statistics

@Domain.Statistics Flow Mean() μ = Σxᵢ/N @Domain.Statistics Flow SampleMean() x̄ = Σxᵢ/n @Domain.Statistics Flow Variance() σ² = Σ(xᵢ - μ)²/N @Domain.Statistics Flow SampleVariance() s² = Σ(xᵢ - x̄)²/(n-1) @Domain.Statistics Flow StandardDeviation() σ = √(Σ(xᵢ - μ)²/N) @Domain.Statistics Flow ZScore() z = (x - μ)/σ

Inferential Statistics

@Domain.Statistics Flow StandardError() SE = σ/√n @Domain.Statistics Flow ConfidenceInterval() CI = x̄ ± z·(σ/√n) @Domain.Statistics Flow TStatistic() t = (x̄ - μ)/(s/√n) @Domain.Statistics Flow ChiSquare() χ² = Σ(O - E)²/E

Correlation & Regression

@Domain.Statistics Flow Correlation() r = Σ(xᵢ-x̄)(yᵢ-ȳ) / √(Σ(xᵢ-x̄)²·Σ(yᵢ-ȳ)²) @Domain.Statistics Flow LinearRegression() ŷ = a + bx b = Σ(xᵢ-x̄)(yᵢ-ȳ) / Σ(xᵢ-x̄)²

Probability

@Domain.Statistics Flow BayesTheorem() P(A|B) = P(B|A)·P(A)/P(B) @Domain.Statistics Flow ExpectedValue() E[X] = Σxᵢ·P(xᵢ) @Domain.Statistics Flow BinomialProbability() P(X=k) = C(n,k)·pᵏ(1-p)ⁿ⁻ᵏ @Domain.Statistics Flow PoissonProbability() P(X=k) = (λᵏe⁻λ)/k! @Domain.Statistics Flow NormalDistribution() f(x) = (1/(σ√(2π)))·e^(-(x-μ)²/(2σ²))
