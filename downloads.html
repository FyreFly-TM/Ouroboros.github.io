Installation

Step 1: Download

  1. Go to the GitHub Releases page
  2. Download the latest .zip file
  3. Extract the zip to a location of your choice

Step 2: Set Up Your Project

  1. Create a new folder for your project
  2. Copy chimera.exe into your project folder
  3. Create your .ouro script files in the same folder

Step 3: Run Your Code

Open a terminal in your project folder and run:

// Run a specific script .\chimera.exe yourscript.ouro // Run the entire project (all .ouro files) .\chimera.exe

Editor Support

VS Code Extension

Syntax highlighting and language support for Ouroboros files.

The extension (.vsix file) is included in the release package.

Installation:

  1. Open Visual Studio Code
  2. Go to Extensions (Ctrl+Shift+X)
  3. Click the three dots (⋯) in the top right corner
  4. Select "Install from VSIX..."
  5. Navigate to and select the .vsix file from the downloaded package

Features

  • Syntax highlighting for .ouro, .var, .comp, .ent files
  • Arrow operator highlighting
  • Keyword and flow detection
  • Custom color theme
