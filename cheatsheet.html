üì¶ Variables

Mutable Variable :=

Can be changed later with =

count := 0 name := "Player" health := 100.0 active := true

Constant =:

Cannot be changed after declaration

PI =: 3.14159 MAX_HEALTH =: 100 TITLE =: "My Game"

Assignment =

Change existing mutable variable

health := 100 // declare health = 75 // update health = 50 // update again

Reactive Binding <-

Auto-updates when dependencies change

width := 10 height := 5 area <- width * height // area auto-updates!

üåä Flows (Functions)

Basic Flow

Flow Greet(name) "Hello, " + name -> Print return true Greet("World")

Flow with Return

Flow Double(n) result := n * 2 return result answer := Double(21) // answer = 42

Multi-Return

Flow Divide(a, b) quotient := a / b remainder := a % b return quotient, remainder q, r <- Divide(17, 5) // q = 3.4, r = 2

Implicit Binding

Return names become available in caller scope

Flow Stats(x, y) sum := x + y product := x * y return sum, product Stats(5, 3) // sum and product now exist!

Arrow Chaining

Flow AddOne(n) return n + 1 Flow Double(n) return n * 2 // Chain flows with -> 5 -> AddOne -> Double -> Print // Output: 12

Recursion

Flow Factorial(n) if n ‚â§ 1 return 1 return n * Factorial(n - 1) Factorial(5) -> Print // Output: 120

üìê Common Patterns

Event Broadcasting

Use: Send one event to multiple handlers
"user_login" -< Logger, Analytics, UI, DB

Data Pipeline

Use: Transform data through stages
rawData -> Parse -> Validate -> Save

Model-View Sync

Use: Keep UI in sync with data
playerHealth <-> healthBar // Either change updates both

Reactive Dashboard

Use: Auto-updating computed values
price := 100 quantity := 5 total <- price * quantity // total updates when either changes

Sensor Aggregation

Use: Combine multiple inputs
temp, humidity, pressure >- AlertSystem

Sequential Processing

Use: Steps must complete in order
Init(), LoadConfig(), StartServer(), Log("Ready")

Parallel Processing

Use: Independent tasks run together
FetchUsers() : FetchProducts() : FetchOrders()

Early Exit

Use: Stop searching when found
for i = 0, i < 1000, i++ if items[i] == target "Found at " + i -> Print kill
