Getting Started

From zero to running code in minutes.

1. Installation

Ouroboros is a compiled language. You'll need the chimera compiler.

  1. Go to the Downloads page.
  2. Download the latest chimera.exe for your platform.
  3. Place it in a folder included in your system's PATH, or keep it in your project folder.

2. Hello World

Create a new file named hello.ouro and open it in your text editor.

In Ouroboros, data flows from left to right. To print text, we send a string into the Print flow.

// hello.ouro "Hello, Ouroboros!" -> Print

3. Your First Flow

Flows are the building blocks of Ouroboros (similar to functions in other languages). Let's define a flow that adds two numbers.

// math.ouro // Define a flow named 'Add' that takes inputs 'a' and 'b' Flow Add -> a, b result := a + b return result // Call the flow sum <- Add -> 10, 5 // Print the result "The sum is: " -> Print sum -> Print

Notice the syntax:

  • Flow Name -> params defines the flow.
  • := assigns a value to a variable.
  • <- (Pull Arrow) captures the return value.
  • -> (Forward Arrow) passes arguments to the flow.

4. Running Code

Open your terminal or command prompt and run the compiler with your script file:

chimera.exe hello.ouro

You should see the output:

Hello, Ouroboros!
