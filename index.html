What makes Ouroboros different?

  • Flow blocks that return multiple values without ceremony.
  • Directional arrows for dataflow: forward, pull, and bidirectional sync.
  • Composable declarations for variables, components, and entities.
  • Lean syntax: indentation-driven blocks, inline formulas, reactive bindings.

Quickstart Highlights

Flows & Returns

Define a flow, pass inputs via a forward arrow, and pull its multiple outputs in one line.

Flow AddAndScale -> a, b sum := a + b scaled := sum * 2 return sum, scaled result, doubled <- AddAndScale -> 5, 7
Reactive Pull

Pull arrows bind a target to a source or expression; updates propagate.

base := 12 perimeter <- base * 4 base = 15 // perimeter now 60
Sync State

Bidirectional sync keeps values aligned—ideal for UI, physics, or gameplay state.

left := 1 right := 2 left <-> right right = 99 // left now 99

Language at a glance

Dataflow mindset

Use forward arrows to push, pull arrows to bind reactively, and sync arrows to mirror state.

Multi-return flows

Types as files
