Key Codes Reference

Complete listing of keyboard and mouse input codes for use with WindowGetKey() and mouse functions.

Usage

Use these codes with the window input functions to detect keyboard and mouse input:

// Keyboard input if WindowKeyPressed(win) key := WindowGetKey(win) if key == 27 // Escape running = false if key == 32 // Space Jump() if key == 65 // 'A' key MoveLeft() if key == 68 // 'D' key MoveRight() // Mouse input if WindowMouseClicked(win, 1) // Left click HandleClick() if WindowMouseClicked(win, 2) // Right click ShowMenu() if WindowMouseClicked(win, 4) // Middle click Pan()

Mouse Buttons

Mouse button codes used with WindowMouseClicked(), WindowMouseDown(), and WindowMouseReleased().

Debug/Menu Example
